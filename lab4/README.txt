################################# LAB 4 README #################################

To run this lab simply open index.html in a browser (I developed it in Chrome).
To move the light you use:

    a: Move light left
    d: Move light right
    s: Move light down
    w: Move light up
    x: Move light forward
    z: Move light backward

    <up arrow>:   Turn light intensity up
    <down arrow>: Turn light intensity down
